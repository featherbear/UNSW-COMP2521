Or, you can copy down the dryruns from /web/cs2521/19T0/assignments/textbuffer/dryrun/ –

cp /web/cs2521/19T0/assignments/textbuffer/dryrun/dryrun{00..10}.c .

To add these to your Makefile, add these magical incantations to the end.

dryruns = $(addprefix dryrun, $(shell seq -w 0 10))
all: ${dryruns}
$(foreach d, ${dryruns}, $(eval ${d}: ${d}.o textbuffer.o))

(For more on what’s going on here, see the GNU Make manual. It’s a pretty good introduction to the syntax and otherworldly magics of Make. In particular, see §8.3 for addsuffix, §8.5 for foreach, and §8.9 for eval.)
